一、Node.js 环境检查
node -v              # 查看 Node.js 版本
npm -v               # 查看 npm 版本

二、项目启动（本地开发）
cd D:\PM_Master_项目源码\app   # 进入项目目录
npm install                     # 安装依赖
npm run dev                     # 启动开发服务器
npm run build                   # 生产构建
npm run preview                 # 预览生产构建

三、Git 版本控制
git init                                                    # 初始化仓库
git add .                                                   # 添加所有文件
git commit -m "init: PM Master 项目管理知识学习平台"        # 提交代码
git remote add origin https://github.com/hema777i/pmhub.git # 连接远程仓库
git push -u origin main                                     # 推送到 main 分支

# 如果推送报错，强制推送：
git branch -M main
git push -u origin main --force

# 后续日常修改：
git add .
git commit -m "xxx"
git push

四、GitHub Pages 部署
npm install -g gh-pages   # 安装 gh-pages 工具
gh-pages -d dist          # 部署 dist 目录

五、浏览器访问
http://localhost:5173              # 本地开发预览
https://hema777i.github.io/pmhub/  # GitHub Pages 线上地址

cd D:\PM_Master_项目源码\app
